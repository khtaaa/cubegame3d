作品名　YOKERO
作成者　加幡　祐人


・起動方法　buildファイルのcybegame3d.exeを起動してください
・プログラム　Assetsファイルのscriptファイルに入っているのでMonoDevelop、Visual Studioなどで起動してください

このゲームは現在作成中のゲームの3Dの陣取りゲームです新しい陣取りゲームを作ろうと思ったとき
このゲームが思いつきました。このゲームでは360度すべてがバトルフィールドになっていて壁や天井なども
移動できるようになっています。このゲームで苦労したことは壁を移動するときのプレイヤーの回転に悩みました
自分で調べたり教師にヒントをいただいたりなどして今の形に仕上げることができました。今後このゲームは
マルチプレイ機能、スキル、ステージのギミックなどをなどを取り入れようと考えています。
